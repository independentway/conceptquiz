## 🚀 Installation and Usage (For Users)

If you simply want to use the program on Linux without dealing with code or terminals, follow these steps:

1. Go to the **https://github.com/independentway/conceptquiz/releases)** section on the right side of this page.
2. Download the file `CONCEPT_QUIZ` from the latest release on your computer (is just an executable file).
3. Double-click the file (or run it from the terminal using `./CONCEPT_QUIZ`).
